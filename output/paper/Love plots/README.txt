This directory contains the Love plots for all the regression models described in the manuscript. Each Love plot presents the exposure-covariate Pearson correlations before and after applying the estimated balancing weights. Specifically, the directory contains the following documents:

* loveplot_rq1.pdf: Love plots of EDC-covariate correlations for the outcome models `attentional function ~ EDC`.
* loveplot_rq2.pdf: Love plots of EDC-covariate correlations for the outcome models `metabolite ~ EDC`.
* loveplot_rq3.pdf: Love plots of metabolite-covariate correlations for the outcome models `attentional function ~ metabolite`.
* loveplot_sex_rq1sa.pdf: Love plots of EDC-covariate correlations for the outcome models `attentional function ~ EDC * sex`.
* loveplot_sex_rq2sa.pdf: Love plots of EDC-covariate correlations for the outcome models `metabolite ~ EDC * sex`.
* loveplot_sex_rq3sa.pdf: Love plots of metabolite-covariate correlations for the outcome models `attentional function ~ metabolite * sex`.
